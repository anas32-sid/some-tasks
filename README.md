# Vector Folder

This folder contains C++ programs demonstrating operations on vectors.

## Programs included

1. **Basic push_back and iteration** – Adding elements to vector and printing them.
2. **Erase elements** – Removing elements from vector using iterators.
3. **Modify elements** – Updating elements based on conditions (e.g., doubling, changing negatives to positives).
4. **Vector capacity and size** – Observing how size and capacity change after push_back.
5. **Pair and vector** – Using vectors of pairs for storing structured data.
6. **Reverse iteration** – Using `rbegin()` and `rend()` to iterate vectors in reverse.
7. **Conditional operations** – Applying discounts, marking even/odd, etc.

Each program has inline comments explaining its functionality.
